PWA ICONS NEEDED
================

Create the following PNG icons for your PWA:

Required:
- icon-192x192.png (192x192 pixels)
- icon-512x512.png (512x512 pixels)
- favicon-32x32.png (32x32 pixels)
- favicon-16x16.png (16x16 pixels)
- apple-touch-icon.png (180x180 pixels)

Optional (for full PWA support):
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-384x384.png

Tip: Use a single 512x512 source image and resize it for all sizes.
Tools like https://realfavicongenerator.net can generate all sizes automatically.
